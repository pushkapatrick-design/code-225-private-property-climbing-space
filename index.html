⚡ Code 225

Private Property Climbing Space Obstruction

Reference Image Climbing space obstruction example
When to Use Code 225
Use Code 225 to report climbing space infractions that are caused by private property owners. This code specifically addresses obstructions originating from private property that interfere with the required climbing space.
Include pictures and comments
When reporting Code 225, include pictures and comments with your report.
Don't use where quadrant climbing space is available
